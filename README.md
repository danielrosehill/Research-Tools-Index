## Templates & Workspaces

### AI Project Spec Pattern

A modular pattern for turning a single high-level plan into a set of per-module specs that a coding agent can execute against — useful for kicking off larger AI-assisted builds.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/AI-Project-Spec-Pattern)

### Claude Code Lawyer

Claude Code workspace template tailored for legal research workflows — document review, case-law digestion, and structured note-taking.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Code-Lawyer)

### Claude Deep Research Template

The baseline Claude Code template for the "deep research" pattern: scaffolds the directory layout (prompts, sources, outputs) used across most research workspaces here.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Deep-Research-Template)

### Claude Preparedness Planner

Workspace template for preparedness and contingency planning research — scenario mapping, checklists, and long-running plan refinement.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Preparedness-Planner)

### Claude Research Space Public Template

Variant of the deep-research template wired for **public publishing** — layout, metadata, and structure geared towards findings that will be shared (vs. kept private).

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Research-Space-Public-Template)

### Claude Stack Research Workspace

A *hands-on* workspace (not a template) for researching and evaluating specific technology stacks — decisions, comparisons, and stack selection notes live here.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Stack-Research-Workspace)

### Claude Tech Research Team

A *template* modelling a team of Claude Code agents for technology and stack research — the reusable scaffold that workspaces like "Stack Research" are built from.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Tech-Research-Team)

### Claude Think Tank

Model repo for a multi-agent "think tank" running inside Claude Code — agents collaborate on research and policy proposal ideation.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/Claude-Think-Tank)

### General Agent Workspace Template

General-purpose file scaffold for human/AI agent collaboration — the broader starting point when a task doesn't fit a research-specific template.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/danielrosehill/General-Agent-Workspace-Template)
